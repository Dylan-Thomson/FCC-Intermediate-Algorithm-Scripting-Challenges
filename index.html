Intermediate Algorithm Scripting

Output is done via console.log in the developer console (try hitting F12)

Please feel free to test the functions on your own by calling them in the console!

sumAll([a, b])
Take array of two numbers. Return sum of those two numbers and all numbers between them.
diffArray(arr1, arr2)
Take two arrays, return the symmetric difference of the two arrays. In other words, return an array of items that appear in one of the arrays but not both.
convertToRoman(num)
Take a number, return its Roman Numeral.
whatIsInAName(collection, source)
Look through an array of objects (collection), returns an array of all objects that have matching property value pairs (source).
myReplace(str, before, after)
Perform a search and replace on the sentence using the arguments provided and return the new sentence.
