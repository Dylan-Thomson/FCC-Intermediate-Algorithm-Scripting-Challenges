Intermediate Algorithm Scripting

Output is done via console.log in the developer console (try hitting F12)

Please feel free to test the functions on your own by calling them in the console!

sumAll([a, b])
Take array of two numbers. Return sum of those two numbers and all numbers between them.
diffArray(arr1, arr2)
Take two arrays, return the symmetric difference of the two arrays. In other words, return an array of items that appear in one of the arrays but not both.
convertToRoman(num)
Take a number, return its Roman Numeral.
whatIsInAName(collection, source)
Look through an array of objects (collection), returns an array of all objects that have matching property value pairs (source).
myReplace(str, before, after)
Perform a search and replace on the sentence using the arguments provided and return the new sentence.
translatePigLatin(str)
Takes the first consonant (or consonant cluster) of an English word, moves it to the end of the word and suffixes an "ay". If a word begins with a vowel you just add "way" to the end.
pairElement(str)
The DNA strand is missing the pairing element. Take each character, get its pair, and return the results as a 2d array. Base pairs are a pair of AT and CG.
fearNotLetter(str)
Find the missing letter in the passed letter range and return it. If all letters are present in the range, return undefined.
booWho(bool)
Check if a value is classified as a boolean primitive. Return true or false.
uniteUnique(arr)
Write a function that takes two or more arrays and returns a new array of unique values in the order of the original provided arrays. In other words, all values present from all arrays should be included in their original order, but with no duplicates in the final array.
convertHTML(str)
Convert the characters &, <, >, " (double quote), and ' (apostrophe), in a string to their corresponding HTML entities.
spinalCase(str)
Convert a string to spinal case. Spinal case is all-lowercase-words-joined-by-dashes.
sumFibs(num)
Given a positive integer num, return the sum of all odd Fibonacci numbers that are less than or equal to num.
